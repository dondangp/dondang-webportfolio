UTA Hackathon Website

Created the legacy database for displaying admin roles and hackathon participants and implement user hooks for role changes on the UTA Hackathon Website using Next.js, I first designed a legacy user hook that served as the foundation for handling user-related actions. Then, I designed the old user table schema, defining the structure of the database to store user information, including fields like user ID, username, email, and role. Using Next.js API routes, I developed server-side endpoints to manage user data, allowing for CRUD operations. To enable users to change roles, I implemented a user hook that handled role transitions when users interacted with buttons associated with each user's data. This hook facilitated role updates via API requests, ensuring that only authorized users could initiate role changes.
